# hello.py

## Overview

`hello.py` is a simple Python script that prints a greeting message to the console. It is often used as a basic example for learning Python syntax and running Python programs.

## Usage

To run the script, use the following command in your terminal:

```sh
python hello.py
```

## Example Output

```
Hello, World!
```

## Code Example

```python
# hello.py
print("Hello, World!")
```

## Customization

You can modify the greeting message by changing the string inside the `print()` function.

```python
print("Hello, Python!")
```
