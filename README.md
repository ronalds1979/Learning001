# Repository Template - Automation Scripts

This repository is a reusable template for automation-script projects.
It provides governance, security, and documentation scaffolding that can be adapted to
any product area.

Use placeholders and replace them for your project:

- `<ORG_NAME>`
- `<PROJECT_NAME>`
- `<PRODUCT_NAME>`
- `<PRODUCT_AREA>`
- `<SUB_PRODUCT>`
- `<WORK_ITEM_ID>`

## Repository Structure (Short)

- `.github/` - Workflows, issue/PR templates, governance
- `docs/` - Standards and documentation
- `product-area/sub-product/` - Product scripts by function
- `shared/` - Shared modules/templates (optional)
